== Description ==

This plugin allows for the use of the RingCentral communication tools platform based on the RingCentral PHP API.

This plugin makes calls to: https://ringcentral.github.io/ringcentral-embeddable-voice/adapter.js

The license is here: https://github.com/ringcentral/ringcentral-js-widgets/blob/master/LICENSE

= Features =

<li>SMS Consent Management within WooCommerce -
This plugin allows for the management of SMS communications authorization and consent from customers of the installed
WooCommerce shop. It will allow for sending SMS messages to customers who have given their consent. It will be
coordinating with the RingCentral eCommerce app.
</li>

<li>SMS Consent checkbox -
An SMS consent checkbox is added to the WooCommerce shop check out page when a shop customer is going through the
checkout process and providing their overall contact and shipping information. This can be turned on/off each time
a customer is completing a purchase from the store.
</li>

<li>A new database table is created -
A new table is created when this plugin is installed to control the consent per customer.
</li>

</ul>

== Installation ==
Use WordPress' Add New Plugin feature, searching "WooCommerce SMS Consent", or download the archive and:
1. Unzip the archive on your computer
2. Upload `woocomm-sms-consent` directory to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
